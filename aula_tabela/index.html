Aula 3 - Tabelas no HTML!

Nesta aula, veremos sobre tabelas do html:

  • Para "iniciar" a tabela, usamos a tag: <table></table>
  • Para colocar os cabeçalhos, usamos a tag: <th></th> (table header)
  • Para colocar cada linha, usamos: <tr></tr> (table row)
  • Dentro de cada linha, mostramos a informação de cada coluna com a tag: <td></td> (table data)
